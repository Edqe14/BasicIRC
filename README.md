# BasicIRC

An basic IRC Server & Client built with Node, Express and Socket.IO

## Installation

First, install all the dependencies: 
```js
npm i
```
Then, run the server/client:

Server
```js
npm start
or
node index.js
```

Client
```js
npm test
or
node client/index.js
```
